# playwrite-pdf-generate-and-download-serverless

#### Prototype for using playwrite and serverless for pdf generation

##### Stack

1. Playwrite
2. Netlify

To install dependencies:

```bash
bun install
```

To run:

```bash
bun dev


```

For running unit test

```sh
bun test
```

To run browser test

```sh
bun test:puppet
```
