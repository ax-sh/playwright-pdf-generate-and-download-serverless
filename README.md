# Playwright-pdf-generate-and-download-serverless

#### Prototype for using Playwright and serverless for pdf generation

##### Stack

1. Playwright
2. Netlify

To install dependencies:

```bash
bun install
```

To run:

```bash
bun dev
```

For running unit test

```sh
bun test
```

To run browser test

```sh
bun test:puppet
```
